## How to compile on linux
- Install [CMake](https://cmake.org/download/) if not done.
- Install uuid if not done
```
$ sudo apt-get install uuid-dev 
```
- Run the application
```
$ mkdir build
$ cd build
$ cmake ..
$ make
$ ./../bin/app
```
